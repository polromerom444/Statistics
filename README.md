Estadística (Grau en Matemàtiques - FME / UPC)

Aquest repositori conté les pràctiques de laboratori, codi en R i resolució de problemes de l'assignatura d'Estadística de la UPC.

## Contingut del repositori

* **Laboratoris (`Rlab0` a `Rlab6`):** Codi amb el que hem anat fent a les sessions de pràctiques: distribucions de probabilitat, manipulació i estadística descriptiva, mètodes de *bootstrap* (i intervals de confiança) i estadística no paramètrica (NP).
* **Dades (`.RData`):** Els datasets utilitzats a classe (Palmer Penguins, DeliveryData...).
